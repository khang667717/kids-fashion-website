chọn 2-core  vì có đư ợc nhiều thời gian sử dụng codespaces hơn) -> 
click Create codespace.

7. Cấp quyền cho Maven Wrapper: chmod +x mvnw  
8. Cài đặt MYSQL: sudo apt install -y mysql-server  
9. Khởi động MYSQL: sudo systemctl start mysql  | sudo service mysql 
start  
10. Kiểm tra MYSQL đã kh ởi động hay chưa: sudo service mysql 
status (nếu hiển thị như ảnh dưới là đã kh ởi động thành công)  
 
11. Tạo database cho MYSQL b ằng câu lệnh truy vấn trực tiếp trên 
terminal:  
sudo mysql -e "CREATE DATABASE IF NOT EXISTS thuchanh6 ;" (LƯU 
Ý: thay đổi thuchanh6 b ằng tên database b ạn muốn tạo) 
12. Kiểm tra database đã t ạo thành công hay chưa: sudo mysql -e 
"SHOW DATABASES;"  
 
13. Tạo tài khoản và cấp quyền để sử dụng database ngoài root:  
sudo mysql -e "CREATE USER IF NOT EXISTS ' user1'@'localhost' 
IDENTIFIED BY '123456';" (bạn có thể thay user1 = tên bất kỳ, 123456 = 
mật khẩu)  
 
sudo mysql -e "CREATE USER IF NOT EXISTS 'user2'@'localhost' 
IDENTIFIED BY '123456';"
